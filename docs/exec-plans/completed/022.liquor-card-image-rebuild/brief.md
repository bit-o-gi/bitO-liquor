# 022.liquor-card-image-rebuild brief

## 배경
- 기존 일괄 생성 이미지는 병 정체성이 약하고, 파일명도 위스키명과 연결되지 않는다.
- 사용자는 카드용 이미지가 각 위스키와 최대한 동일하게 보이고, 파일명도 위스키 네이밍과 유사하길 요청했다.

## 목표
- 상품 원본 이미지를 기준으로 위스키 병 정체성이 유지되는 카드 이미지를 다시 제작한다.
- Storage object filename을 위스키 제품명 기반 slug로 바꾼다.
- 소수 샘플을 먼저 교체하고 화면으로 검증한 뒤 전체 반영한다.

## 범위
- `public.liquor.image_url` 재갱신
- `whisky-images` 버킷 내 카드 이미지 재업로드
- 재현 가능한 제작 스크립트 추가
