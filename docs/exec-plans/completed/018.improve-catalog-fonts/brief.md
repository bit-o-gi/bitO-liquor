# 018.improve-catalog-fonts / brief

## 배경
- 사용자 피드백에 따라 현재 카탈로그의 폰트 인상이 거칠고 브랜드 톤에 비해 덜 세련되게 느껴진다.
- 현재 전역 폰트는 시스템 fallback 위주라 한글/영문 조합에서 에디토리얼 톤이 충분히 살아나지 않는다.

## 목표
- 카탈로그의 한글/영문 타이포그래피를 더 고급스럽고 읽기 좋게 개선한다.
- 브랜드명, 메타데이터, 검색 UI가 새 폰트 조합에서도 자연스럽게 보이도록 맞춘다.

## 범위
- `frontend/app/layout.tsx`
- `frontend/app/globals.css`
- 실행 기록 문서

## 비범위
- 레이아웃 구조 변경
- 카드 정보 구조나 검색/목록 동작 변경
