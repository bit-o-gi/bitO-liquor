# Lean PRD

## 문제
최근 추가한 주류 상세 페이지와 카탈로그 상세 진입 동선이 현재 제품 방향과 맞지 않는다.

## 목표
- 주류 상세 페이지 라우트와 관련 서버 조회 로직을 제거한다.
- 카탈로그 카드는 상세 이동 없이 판매처 가격 비교와 외부 구매 링크에 다시 집중한다.
- 현재 제품 계약 문서를 상세 페이지 미사용 상태에 맞게 정리한다.

## 비목표
- 카탈로그 가격 비교 UX 자체를 재설계하지 않는다.
- 새로운 대체 상세 화면이나 모달을 추가하지 않는다.
- 백엔드 적재/데이터 계약을 변경하지 않는다.

## 성공 기준
- `frontend/app/liquors/[id]` 라우트가 제거된다.
- 카탈로그 카드에서 상세 진입 링크/버튼이 사라지고 판매처 외부 링크만 남는다.
- 현재 동작을 설명하는 문서가 상세 페이지 의존 없이 일관되게 갱신된다.
