# DESIGN

## 현재 방향
- 카탈로그 디자인의 북극성은 "정보 밀도를 유지한 에디토리얼형 주류 카탈로그"다.
- 박스형 SaaS 화면보다, 따뜻한 톤과 넓은 여백, 레이어감 있는 표면 구성이 우선이다.

## 실무 원칙
- 선보다 톤과 간격으로 구역을 나눈다.
- 디자인 실험은 검색/목록/판매처 비교 기능 회귀를 만들면 안 된다.
- 카드, 오버레이, 헤더는 데스크톱과 모바일 상호작용 차이를 인정하고 설계한다.
- 작은 polish는 전역 품질 개선에 흡수하고, 장기 디자인 원칙만 이 문서에 남긴다.
